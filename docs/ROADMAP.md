# Marketing-OS SEO Roadmap

## v0.1 (CLI, 2026年7-8月)

### Core Features

- **LLMO/AEO Audit** — Score how well a website is positioned for AI search engines (ChatGPT, Perplexity, Google AI Overviews)
- **Technical SEO Audit** — Core Web Vitals, structured data validation, sitemap, robots.txt
- **Content Brief Generator** — LLM-powered prompts and brief templates
- **Keyword Intent Mapper (Core)** — Intent classification + LLM clustering + GSC integration

### Out of Scope (v0.1)

- Keyword volume estimation (deferred to v0.2)
- Competitor analysis (deferred to v0.3)
- SERP rank tracking (deferred to v0.3)

## v0.1 (Web UI, 2026年9-10月)

- Web-based dashboard for the CLI features
- Hosted on Cloudflare Pages
- OAuth with Google (for GSC integration)

## v0.2 (2026年Q4 - 2027年Q1)

- Keyword volume estimation
- Multi-language support expansion
- Improved LLMO scoring with more AI models

## v1.0 (2027年Q1-Q2)

- Stable API
- Production-ready integrations
- Marketing-OS commercial product integration

## Future

- Competitor analysis
- SERP rank tracking
- Backlink analysis (potentially)
